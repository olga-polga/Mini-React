## Synopsis

Minimal React js app to deploy to PCF.

## Prerequisites:

install node, npm locally

## Installation
```
npm install  //installs all packages required by the application
```

## Run locally
```
npm run dev
```

## Build
```
npm run build
```

## Tests
```
npm test
```

## Deploy to PCF  (must build first, edit manifest.yml as needed)
```
cf push -p public
```
